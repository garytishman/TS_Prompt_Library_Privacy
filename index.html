Privacy Policy

Tishman Prompt Library — Chrome Extension  •  Effective Date: July 13, 2026

Tishman Prompt Library is an internal browser extension built by Tishman Speyer for use by Tishman Speyer employees and contractors. It lets signed-in users browse, copy, and insert prompts from the organization's internal SharePoint prompt library into supported AI assistant websites. This policy explains exactly what information the Extension accesses, how it is used, where it is stored, and who it is shared with.

Summary

The Extension does not operate any backend server and does not sell, rent, or share any data with third parties for advertising or analytics purposes. All data the Extension touches either stays on your device or is exchanged directly, over HTTPS, with your organization's own Microsoft 365 tenant (Microsoft Entra ID / Microsoft Graph / SharePoint Online) — the same Microsoft services your organization already uses for sign-in and file storage.

Information We Collect

Authentication information

When you click "Sign in with Microsoft," the Extension obtains a short-lived OAuth access token from your organization's Microsoft Entra ID, using Chrome's built-in chrome.identity sign-in flow. This token is used only to call Microsoft Graph on your behalf (to read your department and the prompt library). It is stored solely in chrome.storage.session, which is automatically cleared when you close your browser or sign out, and it is never transmitted anywhere other than Microsoft's own sign-in and Graph API endpoints.

Personally identifiable information

The Extension reads your Microsoft Entra ID department attribute (via Microsoft Graph's /me endpoint) solely to determine which department-specific prompt list to display alongside the shared "Common" list. This value is cached only in chrome.storage.session for the duration of your browser session and is never transmitted to any server other than Microsoft Graph. The Extension does not collect your name, email address, phone number, or any other personal identifiers beyond this single attribute.

Organizational content

The Extension fetches prompt entries (title, description, tags, author, and last modified date) from your organization's SharePoint prompt library via Microsoft Graph, so they can be displayed and searched inside the Extension. This content belongs to your organization, not to any individual user, and is cached locally in chrome.storage.session to avoid repeated network requests within the same browser session.

Local preferences

The Extension stores one small preference — your light/dark theme choice — in chrome.storage.local so it persists across browser restarts. This is a UI setting only and is never transmitted anywhere.

Website interaction (AI assistant tabs)

When you click "Use" on a prompt, the Extension checks the hostname of your currently active browser tab (only at that moment, via the activeTab permission) to confirm you're on a supported AI assistant site (ChatGPT, Claude, Microsoft Copilot, or Bing). If so, it inserts the selected prompt's text into that page's input box on your behalf. This is a one-way action: the Extension writes prompt text into the page; it does not read, collect, or transmit any content, conversation, or data from those AI assistant sites.

What we do not collect

  • Location data: not collected.
  • Browsing history: not collected, logged, or transmitted. The Extension only checks the active tab's hostname at the moment you click "Use," and never stores or reports it.
  • Financial or payment information: not collected.
  • Health information: not collected.
  • Analytics, tracking, or advertising data: not collected. The Extension includes no analytics SDKs, trackers, or advertising libraries of any kind.

Permissions Used

The Extension requests the following Chrome permissions, each used strictly for the purpose described:

  • identity — to sign in with your Microsoft work account via Chrome's OAuth flow.
  • storage — to cache your session token, prompt data, and theme preference locally in your browser.
  • activeTab — to detect, only when you click "Use," whether your current tab is a supported AI assistant site.
  • scripting — to insert the selected prompt text into that page's input field.
  • Host permissions for login.microsoftonline.com, graph.microsoft.com, chatgpt.com, chat.openai.com, claude.ai, copilot.microsoft.com, and bing.com — required to sign in, retrieve prompts from your organization's Microsoft 365 tenant, and insert prompt text into these specific AI assistant sites.

Where Data Is Stored and How It's Transmitted

All data the Extension handles is stored locally in your browser (chrome.storage.session or chrome.storage.local) — Tishman Speyer operates no backend server for this Extension, and no data is ever transmitted to or stored on any infrastructure controlled by the developer. The only network requests the Extension makes are direct, encrypted (HTTPS/TLS) calls from your browser to Microsoft's own services: Microsoft Entra ID (for sign-in) and Microsoft Graph (for reading your department and the SharePoint prompt library). These are the same Microsoft 365 services your organization already relies on for identity and file storage.

Third-Party Services

The Extension communicates only with Microsoft Entra ID and Microsoft Graph / SharePoint Online, which are part of your organization's existing Microsoft 365 tenant and governed by your organization's existing agreements with Microsoft — not a separate third party introduced by this Extension. The Extension integrates with no other third-party service, and includes no analytics, crash-reporting, or advertising providers of any kind.

Data Sharing

We do not sell, rent, trade, or otherwise share any data collected by the Extension with any third party for advertising, marketing, or any other unrelated purpose.

Data Retention, Access, and Deletion

Session-based data (your sign-in token, cached department, and cached prompt list) is automatically deleted when you close your browser or sign out of the Extension. The only value that persists across sessions is your theme preference, which you can clear at any time via chrome://extensions (remove the Extension, or clear its storage) — this immediately deletes all locally stored data.

Because the Extension does not transmit or store any data on a server operated by the developer, there is no remotely stored personal data for us to provide access to or delete on your behalf. If you have any questions about what the Extension stores on your device, or need help clearing it, contact us using the details below.

Children's Privacy

This Extension is an internal workplace tool intended solely for use by Tishman Speyer employees and contractors. It is not directed at, and is not intended for use by, children under the age of 18.
